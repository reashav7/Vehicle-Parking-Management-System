
<h1>🚗 Vehicle Parking App – Flask Web Application</h1>

<p>This is a web-based <strong>Smart Parking Management System</strong> built using <strong>Flask</strong>, <strong>SQLAlchemy</strong>, and <strong>SQLite</strong>. It supports user registration, parking spot booking, real-time availability, admin management of parking lots and spots, and reservation history.</p>

<hr />

<h2>✨ Features</h2>
<ul>
  <li>👤 User registration & login</li>
  <li>🅿️ Real-time parking spot booking</li>
  <li>📍 Admin control panel to manage parking lots and spots</li>
  <li>⏱ Track start/end time of parking</li>
  <li>💸 Cost calculation based on hours parked</li>
  <li>📊 View reservation history for both users and admins</li>
  <li>🔐 Session-based authentication for both users and admins</li>
</ul>

<hr />

<h2>💠 Tech Stack</h2>
<ul>
  <li><strong>Backend</strong>: Python, Flask, SQLAlchemy</li>
  <li><strong>Frontend</strong>: HTML, CSS, Bootstrap, Jinja2</li>
  <li><strong>Database</strong>: SQLite</li>
  <li><strong>Authentication</strong>: Werkzeug, Flask session</li>
</ul>

<hr />

<h2>🚀 Getting Started</h2>

<h3>1. Clone the Repository</h3>
<pre><code>git clone https://github.com/your-username/parking-management-flask.git
cd parking-management-flask
</code></pre>

<h3>2. Set Up a Virtual Environment</h3>
<pre><code>python -m venv venv
source venv/bin/activate      # On Windows use: venv\Scripts\activate
</code></pre>

<h3>3. Install Required Packages</h3>
<pre><code>pip install -r requirements.txt
</code></pre>

<h3>4. Run the Application</h3>
<pre><code>flask run
</code></pre>
<p>Then open your browser and navigate to <a href="http://localhost:5000">http://localhost:5000</a></p>
